### 2.2 Основная навигация

Структура (слева направо):
1. Логотип: текстовая ссылка **"Dropbox Brand Studio"** → `https://dropbox.com`
2. Знак рукопожатия 🤝 и ссылка **"Daybreak Studio"** → `https://daybreak.studio/`
3. Горизонтальный список пунктов меню:
   - Framework
   - Voice & Tone
   - Logo
   - Typography
   - Iconography
   - Color
   - Imagery

**Поведение навигации:**
- Фиксированная (`position: sticky` или `fixed`) при скролле

## 9. Типографика (глобальные правила)

### 9.1 Основной шрифт

**DB Sharp Grotesk** — кастомный шрифт Dropbox.  
Создан совместно с **Sharp Type** на базе Sharp Grotesk.  
Вариабельный (Variable Font): оси Weight, Width, Contrast.

**Подключение:** через CDN Dropbox или локальный файл шрифта. Формат: WOFF2 (обязательно) + WOFF (fallback).

```css
font-family: 'DB Sharp Grotesk', 'Sharp Grotesk', sans-serif;
```

### 9.2 Размеры заголовков (приблизительные)

| Элемент | Размер | Weight |
|---|---|---|
| Hero / Splash заголовок | 80–120px | 700–900 |
| Section заголовок | 48–64px | 600–700 |
| Подзаголовок | 20–28px | 400–500 |
| Body | 16–18px | 400 |
| Навигация | 14–16px | 400–500 |
| Баннер-цитата | 12–13px | 400 |

### 9.3 Line-height

- Основной: `120%` (1.2) — задан явно в гайдлайнах
- Для крупных заголовков допустимо 1.0–1.1
- Для body-текста: 1.5–1.6

### 9.4 Поведение вариабельного шрифта

На тёмном фоне — использовать чуть более лёгкий weight, чем на светлом (оптическая компенсация). Это явно описано в гайдлайнах к шрифту.

## 11. Анимации — детальное описание

### 11.1 Глобальный easing

```
cubic-bezier(0.65, 0, 0.45, 1)
```

Применяется ко всем переходам. Характер: плавный вход, мягкий выход — не механический, «физический».

### 11.2 Scroll-triggered анимации

**Тип:** IntersectionObserver — элемент начинает анимацию, когда входит во viewport.

**Паттерн для текстовых блоков (поочерёдный вывод слов/строк):**
```js
// Псевдокод
words.forEach((word, i) => {
  word.style.animationDelay = `${i * 80}ms`;
  word.classList.add('is-visible'); // при входе в viewport
});
```

**CSS-базис:**
```css
.word {
  opacity: 0;
  transform: translateY(20px);
  transition: opacity 0.5s cubic-bezier(0.65, 0, 0.45, 1),
              transform 0.5s cubic-bezier(0.65, 0, 0.45, 1);
}
.word.is-visible {
  opacity: 1;
  transform: translateY(0);
}
```

### 11.3 Stagger для карточек/файлов

```css
.card:nth-child(1) { transition-delay: 0ms; }
.card:nth-child(2) { transition-delay: 60ms; }
.card:nth-child(3) { transition-delay: 120ms; }
/* и т.д. */
```

### 11.4 Hover-эффекты

- Навигационные ссылки: `color transition` ~150–200ms
- Карточки файлов: `transform: scale(1.02)` или `box-shadow` ~200ms
- CTA-кнопки: background/border transition ~150ms

### 11.5 Переходы между страницами (page transitions)

- Fade-out текущей страницы → fade-in новой
- Смена фоновой темы (светлая → тёмная) происходит синхронно
- Длительность: ~300–400ms

### 11.6 Видео-блоки

- Все фоновые видео: `autoplay muted loop playsinline`
- Без нативных controls
- `object-fit: cover` в контейнере с фиксированными размерами

## 15. Технические требования к реализации

### 15.1 Обязательно

- Шрифт DB Sharp Grotesk подключается через `@font-face` как variable font (`.woff2`)
- Все анимации реализованы через IntersectionObserver + CSS transitions (без тяжёлых JS-анимационных библиотек, если не критично)
- Видео: `autoplay muted loop playsinline` без controls
- Переключение темы (светлая/тёмная) через CSS custom properties (`--bg`, `--text`, `--accent`) на уровне `:root` или атрибута `data-theme`

### 15.2 Зависимости (опционально)

- **Lottie Web** — для Lottie-анимаций на странице Motion
- **GSAP ScrollTrigger** — если stagger-анимации при скролле реализуются через него (в оригинале Webflow может использовать собственный движок)

### 15.3 Производительность

- Изображения: формат `.webp`, lazy loading для всего ниже fold
- Видео: `preload="none"` или `preload="metadata"`, начало воспроизведения только при попадании в viewport
- Шрифт: `font-display: swap`

## 16. Открытые вопросы / требует уточнения

1. **Точные HEX-коды** всех акцентных цветов и серой шкалы — необходимо снять через DevTools или получить Figma-файл.
2. **Файл шрифта DB Sharp Grotesk** — лицензия кастомная, публично не доступен. Нужно согласовать использование с Dropbox или найти ближайший доступный аналог (Sharp Grotesk от Sharp Type).
